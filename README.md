## Описание проекта

В рамках проекта была разработана модель машинного обучения для предсказания риска сердечного приступа на основе медицинских показателей пациентов и факторов образа жизни.  
Модель обучена на исторических данных пациентов и интегрирована в API сервис, позволяющий автоматически выполнять предсказания для новых данных.

Сервис принимает CSV файл с данными пациентов, выполняет предсказание риска сердечного приступа и возвращает результаты в формате JSON.

## Анализ данных

Датасет содержит **8685 наблюдений и 27 признаков**, описывающих медицинские показатели пациентов, наличие хронических заболеваний и факторы образа жизни.

Целевая переменная (`target`) отражает риск сердечного приступа.

Распределение классов:

- 0 (низкий риск) — 65%
- 1 (высокий риск) — 35%

В данных присутствуют пропуски в ряде признаков. Для корректной работы моделей пропуски были обработаны заполнением значением 0.

Корреляционный анализ показал слабую взаимосвязь между признаками (|corr| < 0.2), что говорит о том, что риск сердечного приступа определяется совокупностью факторов.

---

## Обучение моделей

Для решения задачи классификации были обучены несколько моделей машинного обучения:

- Logistic Regression
- Random Forest
- CatBoost

Данные были разделены на обучающую и валидационную выборки.

Наилучший результат показала модель **Logistic Regression** после балансировки классов и подбора порога классификации.

Итоговые метрики модели:

- **F1-score = 0.52**
- **Recall = 0.52**
- **Precision = 0.37**

## API сервис

Для использования модели был реализован сервис на **FastAPI**.

Endpoint:
POST /predict

POST /predict

API принимает CSV файл с данными пациентов и возвращает предсказания.

Пример ответа:
[
{"id": 7746, "prediction": 0},
{"id": 4202, "prediction": 1},
{"id": 6632, "prediction": 1}
]

## Инструкция по запуску приложения

### 1. Клонирование репозитория

git clone <repository_url>
cd ProjectMonth

### 2. Установка зависимостей

conda activate practicum

### 3. Запуск API сервиса


uvicorn api.app:app --reload


После запуска сервис будет доступен по адресу:


http://127.0.0.1:8000
